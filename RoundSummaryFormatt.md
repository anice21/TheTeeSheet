## Round Summary Formatting Updates

- Display each player's **Total Score**.
- Display **Score Relative to Par** next to the total (e.g., `+2`, `-1`, `E` if even).
- If the score relative to par is negative, display it in **red**.

### Example

**Player Name** — Total: 72 (**E**)
**Player Name** — Total: 74 (**+2**)
**Player Name** — Total: 70 (**-2**)
